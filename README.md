# 💰 Personal Finance Tracker
A simple Python tool to track your income, expenses, and savings rate.

## 📌 Features
- Enter your monthly income and expenses
- Automatically calculates savings and savings rate
- Tells you if your saving habit is on track

## 🚀 How to Run
Enter your income and expenses when prompted

## 📊 Sample Output
Enter monthly income: ₹50000
Enter monthly expense: ₹37000

--- Finance Summary ---
Income:       ₹50,000.00
Expenses:     ₹37,000.00
Savings:      ₹13,000.00
Savings Rate: 26.0%
✅ Great saving habit!

## 🛠️ Built With
- Python 3.x
- Jupyter Notebook (prototyped)
